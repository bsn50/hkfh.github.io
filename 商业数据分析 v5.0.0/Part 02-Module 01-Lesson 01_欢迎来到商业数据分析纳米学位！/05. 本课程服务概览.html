05. 本课程服务概览

是不是已经迫不及待要开始新课程的学习了呢？
为了帮助你能更好地展开学习，请在开始前，仔细阅读「学前必知」的有关内容哦~

本部分不包含核心课程内容，你仍需逐一仔细阅读的原因是：当你学习新技术时一定会遇到困难/问题，那么，你应该提前了解在 Udacity 更高效解决这些问题的方法和渠道。这些内容无疑最适合在一开始就准确了解，让我们开始吧！

在本课程的学习过程中，我们将主要通过 项目审阅，中文论坛 等服务来帮助你在课程有效期内完成本纳米学位。

下面我们就来简单了解一下这些服务：

项目审阅 -- Learning by doing

在你提交项目后，我们将有技术行家来帮助你审阅提交的项目，为你提供细致到每行代码的审阅。 收到评审师反馈后，如果没有达到项目要求，你可以多次提交项目，让项目评审师提供指导，直到你达到项目要求。过程中，你需要注意：

  • 项目审阅标准：每一个项目都会有明确的审阅标准，提交前务必仔细阅读，这样可以提高通过效率。
  • 项目提交时间：为了帮助你规划时间，我们为每一个项目设定了「建议完成时间」，非常建议你按照这个时间通过项目，这样可以帮助你及时毕业；如果超过了这个时间，你也不用太担心，只要课程仍在有效期内，这并不会导致你不能提交项目，你只需要在接下来的时间里多努力一些就可以重新赶上建议进度，同样可以在有效期内顺利毕业。
  • 提交渠道与方式：所有项目均需要在教室内提交项。提交的形式会有所不同，根据项目要求会略有不同。具体为提交本地文件，通过Github提交，或者直接在Workspace中提交，请以各项目具体要求为准。

你可以在「教室使用指南」-「项目提交指引」查看更多关于项目提交的信息。

中文论坛 -- 探索并解决问题

在优达学城，你不是独自一人学习。优达学城导师、Mentor都活跃在论坛，他们无比乐意为你提供帮助。我们也鼓励大家都可以把学习中遇到的技术问题、学习感想沉淀在论坛，不仅有迹可循，亦能帮助他人。你可以前往优达论坛找到你的同学们，有效地提问和参与讨论，将这个知识库打造成能使你和其他同学都能反复获益的平台。

  • 注意收藏网址：论坛地址是：https://discussions.youdaxue.com/。登陆优达账号，该账号需有当前纳米学位的购买记录，即可看到相应板块。
  • 先搜索再提问：当你在学习过程中遇到问题，请随时前往论坛，看看是否已经有人提出过类似问题，或提出你自己的问题，或是回答其他同学的问题。在正式发帖提问前，我们更建议你使用 Google、Stack overflow 或者论坛先进行搜索，这样不仅可以更高效地解决具体问题，并且可以不断增强你的解决问题的能力。 这个指南可以帮助你熟悉如何使用论坛！

你可以在「教室使用指南」-「优达论坛」查看更多关于论坛使用的技巧。

学习支持

当你遇到问题时，如果它是学习上的问题，我们推荐你充分使用 Google、 Stack Overflow 、优达论坛（上述）等搜索引擎或垂直社区；也鼓励你前往相应课程的微信学习社群，与同学、助教等一起交流。

对于其他问题，如果是与账号、课程加入、发票相关的常见问题，你可以先查看我们的常见问题中心，或许可以更快获得答案，也可随时点击此处 提交工单,与我们服务支持团队取得联系。

期待拥有卓群自学能力和自律能力的你，从本课程中顺利毕业。

在「学前必知」的后面两个部分，你可以继续了解到：

  • 「课程注意事项」包含了其他同学会频繁问到的问题，可能也会有你好奇的问题；
  • 「教室使用指南」则详细介绍了教室有关的使用技巧，可以帮助你提高学习效率。
