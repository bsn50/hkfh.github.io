03. 课程概述：3步让你具备数据分析能力

「商业数据分析」纳米学位 课程的设计初衷， 是为了帮助商业领域想要提升数据分析能力的你，在短时间内，高效和系统地掌握数据分析的基础技能。

毕业后，你将可以：

  • 解读数据报表，通过观察数据面板得到数据分析报告
  • 理解并掌握商业数据分析的框架，具备提出问题并通过数据来回答的能力，有能力针对不同的商业问题选择不同的数据分析方法
  • 学会使用 Excel 计算统计参量、绘制可视化图表、搭建简单的机器学习模型进行趋势预测
  • 针对分析对象的不同选择合适的数据可视化类型、评估数据可视化的有效性
  • 使用 SQL 语言查询数据，挖掘业务信息
  • 有能力处理真实世界的数据，即对各种缺失值以及错误输入数据的处理
  • 为成为数据分析师或数据科学家打下坚实基础

Learning by doing 是 Udacity 的教学理念，硅谷教室通过 3 步帮你零基础掌握商业数据分析：

  • Step 1：通过视频学习基础知识
  • Step 2：通过及时的互动练习巩固视频所学知识
  • Step 3：通过实战项目挑战，以及针对实战项目导师答疑、学习辅导，把知识转化成应用能力

通过视频课程的学习，结合 Udacity 独家的模拟真实商业挑战的项目， 你可以在 3 个月内快速从零基础起，学习基本的数据分析技能，掌握高效的分析工具并应用于商业分析领域，实现职业发展的提升。Udacity 在数据科学领域的积累，来自硅谷的教学内容，能够让你掌握强大的数据分析的独门秘方。
