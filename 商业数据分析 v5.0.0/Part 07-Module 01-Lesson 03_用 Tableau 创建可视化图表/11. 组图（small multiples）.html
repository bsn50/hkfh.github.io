11. 组图（small multiples）

小多组图

组图（small multiples）

组图是我最喜欢的可视化技巧之一。在 Tableau 里可以便捷地实现组图可视化，只需将多个维度拖到 Columns 和 Rows 功能区上即可创建组图。

季度小多组图

小多组图 2

图中是四个单独的图表，显示了了四年内每个季度的利润增长情况。注意到图表按水平方向分布，这是由 Columns 工作区中的 Order Date 字段造成的。

但是，是什么因素推动了这些增长？我们来看看。可以将 Category 拖到 Rows 工作区内。这样将创建更多的图表，数据按照 Category 划分并垂直分布。

Order Date 类别的小多组图

小多组图 3

现在可以看出，技术是利润增长的主要推动因素。组图适合展示绝大多数的图表类型，下面是几个示例。

小多组图示例

小多组图示例

在上述散点图中，你可以轻松地对比每个类别每项的利润。现在就很清楚为何技术是增长的主要推动因素了（正如之前看到的），每项的利润要比其他类别的利润高很多。

组图还适用于地图。从上面右图中可以看出哪些州在亏损，哪些州在盈利。
