分析问卷数据

对于最后一个项目，你将自己进行数据分析并创建一个可分享的文件来记录你的发现结果。完成数据分析流程，从提出问题开始，到最终分享你的发现结果。

评估

使用审阅标准检查你的项目。如果你对自己的提交内容满意，那么就可以提交了！如果你觉得在任何方面尚不满足规范，还有待改进，请继续加油！

优达的专业审阅者将根据同一审阅标准评估你的项目。你的项目必须满足每个类别的标准，提交的项目才能通过评估。

提交

提交项目中应该包含的文件

  1. 一个演示文稿文件，该文件应该包括：

    • 关于你提出的问题的陈述
    • 记录你通过电子表格工作簿进行的任何数据清理工作
    • 呈现最终结果的总结性统计信息和图表
    • 利用我们的模板

  2. 列出你在创建提交内容的过程中参考或使用的网站、图书、论坛、博客等（如果没有用到任何此类资料，则填上 N/A）。
