SQL 查询

Criteria Meet Specification

学员编写的 SQL 查询中，没有出现错误。

所有的 SQL 查询在执行时都没有出现错误。

学员可以在 SQL 查询中正确地使用连接。

每个 SQL 查询都需要包含一个或多个连接。

学员可以在 SQL 查询中正确地使用聚合。

每个 SQL 查询都需要包含一个或多个聚合。

学员可以在 SQL 查询中正确地使用子查询。

至少一个 SQL 查询包含子查询。

学员可以通过使用 SQL 回答多个问题。

学员在其提交的项目中至少使用了 4 个唯一 SQL 查询。

演示文稿

Criteria Meet Specification

学员提交的幻灯片组织有序，易于阅读与理解。

提交的幻灯片组织有序，逻辑条理。每张幻灯片都有一个标题，可视化描述中没有严重的拼写错误与语法错误。

学员知道如何创建用于提供有用信息的数据可视化。

所有的可视化都能够提供同一指定维度（比如：客户、供应商、产品、雇员）的一些信息。

学员可以为数据可视化编排清晰的格式，并充分利用标签。

所有可视化都应包括标题与轴标签，并在适当的地方附有图例，且易于理解。

Tips to make your project standout:

为了让你的项目脱颖而出，请参考以下几点建议：

  • 通过数据寻找有趣的模式与见解，而不是简单地进行概要统计。
  • 使用本课程中未讲授的其他高级 SQL 函数，如case函数。
  • 在可视化中充分利用颜色、大小与形状。
  • 利用每张幻灯片上的幻灯片标题和/或图表描述来阐述可视化中的主要见解。
